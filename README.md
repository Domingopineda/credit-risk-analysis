# Credit-Risk-Analysis

## Project Overview
 I analyze key factors of credit risk using SQL for data transformation and tableau for visualization. identify that income, type of loan, interest rate and credit history are determining variables in the probability of default.

## Business Problem 
Financial institutions need to evaluate borrower risk accurately to minimize loan defaults.

## Tools Used
- SQL Server Microsoft 
- Tableau
- Excel

## Key Questions
- Do lower-income applicants have higher default rates?
- Which loan purposes carry the highest risk?
- How does interest rate impact default probability?

## Key Findings 
-people with lower income show more risk and the risk increases a lot with active loans. 
-loans intended for home improvement have a higher rate of non-compliance.
-The higher interest rate is related to a greater probability of default, which says that high risk loans suggest a higher interest rate
